# Track.now Business Value and Monetization Spec

Date: 2026-04-12

## Positioning

Track.now is not competing on feature volume. It is competing on:
- Calm execution
- Daily action clarity
- Emotional sustainability

Core promise: "Do what matters now."

## Primary Revenue Model

## 1) Free
- Max 5 habits
- Today + basic mission tracking
- Core reminder experience

## 2) Pro ($4.99/month target)
- Unlimited habits
- Advanced analytics and momentum
- WhatsApp summary layer
- Social accountability (friends/groups)

## 3) Family ($9.99/month target)
- Multi-member routines
- Household mission layer
- Guardian-focused summaries

## Monetization Triggers in Product

- Habit count limit (free cap)
- Advanced analytics access prompt
- WhatsApp summary toggle prompt
- Family route upgrade prompt

## Funnel Strategy

1. Acquire with clarity:
   - Landing page promise: focused growth without noise.
2. Activate fast:
   - First mission created in onboarding.
3. Build retention:
   - Today-first loop + reminder execution + momentum.
4. Convert:
   - Upgrade when user has active routine and visible value.

## Product Metrics to Track

- D1, D7, D30 retention
- Daily active users
- Active habits per user
- Session completion rate (especially multi-session habits)
- Free-to-Pro conversion
- Pro-to-Family upgrade rate

## Profitability Logic

- Low infra overhead in local/demo + Firebase baseline
- High subscription leverage with strong retention
- Clear expansion potential via group/family loops

## Next Business Milestones

1. Connect real Stripe checkout and webhook backend.
2. Add conversion analytics around upgrade prompts.
3. Launch creator/community challenges for organic acquisition.
